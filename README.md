# python-package

A modern python package template with professional grade best practices.

## Components
- [x] Build backend
	- poetry (chosen to replace make if desired)
- [x] Build tool
	- make
- [x] Dependency management
	- venv
	- requirements.txt
- [x] Linters
	- pylint
- [x] Typing
	- mypy
- [x] Auto-formatters
	- isort
	- autoflake
	- Black
- [x] Documentation
	- MkDocs/MkDocstrings
- [ ] Changelog
- [x] Testing
	- pytest
	- coverage
- [ ] Pre-commit
- [ ] CI/CD
- [ ] LICENSE
	- [Choose a license](https://choosealicense.com/)
- [ ] Logging
- [x] Typing
	- [Static typing with python](https://typing.readthedocs.io/en/latest/)
	- [Python type system](https://typing.readthedocs.io/en/latest/spec/index.html)
	- [mypy](https://github.com/python/mypy)

## Infrastructure
- Ansible
